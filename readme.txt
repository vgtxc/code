imp->文件夹目录及功能说明
根目录
	|-src;			@->资源目录, 存放说明文档与其它非工程文件
	|---|---doc;	@->文档目录, 存放文档说明笔记等
	|---|---img;	@->图片目录, 存放测试图片
	|-work;			@->工作目录, 存放代码与专业软件工程文件
	|---|---lang;	@->程序源码
	|---|---profession;	@->专业软件
	|---|---project;	@->实战测试
	|-tmp;				@->临时目录, 存放临时代码

imp->文件后缀名及说明
	程序源码文件
	|-clang
	|---"*.c":c语言源码
	|---"*.cpp":c++语言源码
	专业工程文件
	|-solidwork
	|---"*.step":零件
	|---"*.sldasm":装配体
